## В данном примере будет использоваться стороннее приложение [**Freesm Launcher**](https://freesmlauncher.org) — удобный лаунчер для настройки и игры в любую версию Minecraft.

> _🚀 Лаунчер имеет ряд плюсов по сравнению с официальным лаунчером Mojang: установка сборок в один клик, гибкая настройка и многое другое._

Freesm доступен как для лицензионного (купленного) Minecraft, так и для неофициального. Также существует [**Prism Launcher**](https://prismlauncher.org/) 💎 — только для лицензионной копии игры.

---

### 1. 📥 Установка лаунчера

- Чтобы установить лаунчер для Windows, перейдите по [**этой ссылке**](https://freesmlauncher.org/ru/downloads) и скачайте нужную вам версию 💾.
- Запустите установщик и установите программу в удобную для вас папку (или откройте портативную версию), а затем запустите лаунчер через `.exe` файл.

![Сайт лаунчера](/guides/minecraft/launcher-site.webp)

- Выберите русский язык 🇷🇺.

**👤 Настройка аккаунта:**

1.  **Если у вас куплен Minecraft:**
    Нажмите **«Войти через Microsoft»** 🟦 и авторизуйтесь в своем аккаунте. В случае успеха у вас должна появиться надпись **«Успешный вход»** ✅.
2.  **Если у вас нет лицензии:**
    Выберите пункт **«Add Offline account»** 🏴‍☠️ и введите никнейм, который будет использоваться в игре.

Перейдите в главное меню лаунчера и приготовьтесь к добавлению игры.

![Выбор входа](/guides/minecraft/account-login.webp)

---

### 2. ➕ Добавление сборки

Чтобы добавить новую версию игры или сборку внутри лаунчера, нажмите **«Добавить экземпляр»** в верхней панели.

![Кнопка "добавить экземпляр"](/guides/minecraft/add-instance-button.webp)

**Варианты создания:**

1.  🍦 Создать ванильную сборку **без модов** (не рекомендуется).
2.  ⚙️ Собрать **свою сборку модов** (на оптимизацию или контент), выбрав загрузчик слева.
3.  🟢 Выбрать **готовый набор модов в Modrinth** (на оптимизацию или контент) — **рекомендуется!**

![Выбор модов](/guides/minecraft/modpack-selection.webp)

> ❗ **Обратите внимание:** сервис Modrinth может не работать в РФ. Рекомендуется использовать прокси 🌐.

**Альтернативный способ:** Можно скачать любую сборку из интернета и добавить её, просто перетащив файл в главное окно лаунчера. 📂➡️🧊

> _Некоторые разработчики не позволяют использовать свои моды в сторонних лаунчерах._
> ![Моды не найдены](/guides/minecraft/mods-not-found.webp)

**Если при загрузке сборки некоторые файлы «недоступны для загрузки», скачайте их вручную:**

- Нажмите **«Открыть недостающие»** 🔗 — откроются все ссылки на скачивание модов.
- Дождитесь загрузки **каждого** файла ⏳.
- По мере скачивания лаунчер сам найдет недостающие моды в папке «Загрузки» и поменяет их статус (цвет) на зелёный 🟢.
  ![Мод найден](/guides/minecraft/mod-found.webp)
- Если ваша папка загрузки отличается от стандартной, нажмите **«Добавить папку загрузки»** 📂.
- Когда все моды станут зелёными, нажмите **«OK»** ✅.

![Все моды найдены](/guides/minecraft/all-mods-found.webp)

### 3. ⚙️ Настройка сборки

- В этом примере будет использоваться сборка на оптимизацию Minecraft — [**SimplyFine**](https://modrinth.com/modpack/sif).
- Введите в поиск 🔍 вверху название сборки, а справа выберите нужную версию Minecraft (например, 1.21.1).
  ![Поиск SimplyFine](/guides/minecraft/simplyfine-search.webp)
- Нажмите **«ОК»** и дождитесь установки.
- **После этого уже можно играть** 🎮, дважды щелкнув по иконке сборки или нажав «Запустить» ▶️. Либо можно настроить сборку более детально, нажав правой кнопкой мыши и выбрав **«Изменить»** 🔧.

#### 3.1. 🧩 Моды

- Перейдите на вкладку **«Моды»**, чтобы добавить, удалить или отключить моды.
- Чтобы добавить новый мод, нажмите **«Скачивание модов»** ⬇️, выберите нужные, для каждого нажмите **«Выбрать мод для загрузки»**, а затем **«Посмотреть и подтвердить»**.
  > ⚠️ _Если вы хотите скачать моды, не подходящие вашей версии игры, отключите «Фильтр опций»._
- Все моды можно проверить на наличие обновлений 🔄 (если, например, вы долго не обновляли сборку). Для этого нажмите кнопку **«Проверить обновления»** в меню модов.
- Можно добавить мод из файла, нажав **«Добавить файл»**, или просто перетащить файл в окно списка модов.
- Удалять ненужный мод необязательно — его можно отключить, сняв галочку ☑️ слева от названия.
- Открыть папку со сборкой в проводнике можно, нажав кнопку **«Открыть папку»** 📂.

> ⛔ **Моды нельзя добавлять, пока игра запущена.**

![Интерфейс выбора модов](/guides/minecraft/mods-interface.webp)

#### 3.2. 📦 Наборы ресурсов (Ресурспаки)

- По аналогии с модами, наборы ресурсов добавляются во вкладке **«Наборы ресурсов»**.
- Чтобы скачать ресурспак, введите название и выберите версию, а затем нажмите **«Выбрать набор»** и **«Посмотреть и подтвердить»**.
- После загрузки ресурспак появится в игре.
  > ✅ _Ресурспаки можно добавлять во время запущенной игры._

#### 3.3. ✨ Шейдеры

- Наборы шейдеров (шейдер-паки) добавляются во вкладке **«Наборы шейдеров»**.
- Чтобы скачать шейдер, введите название и выберите версию, а затем нажмите **«Выбрать набор»** и **«Посмотреть и подтвердить»**.
- Загруженный шейдер появится в настройках игры.
  > ✅ _Шейдеры можно добавлять во время запущенной игры._

### 4. 🚀 Производительность и Java

#### 4.1. ☕ Настройка Java

**По умолчанию Java не оптимизирована для игры в Minecraft. Рекомендуется установить пользовательскую версию — Java-Adoptium.**

- Внутри настроек сборки перейдите в раздел **«Параметры»** → **«Java»**.
- Поставьте галочку **«Установка Java»**.
- Нажмите **«Скачать Java»** ⬇️, если вы не скачивали её ранее.
- Перейдите во вкладку **«Adoptium»** и скачайте последнюю версию (_запомните имя Java_).
  ![Варианты Java](/guides/minecraft/java-adoptium.webp)
- По окончании загрузки нажмите **«Auto-detect»** 🔎 и выберите из списка ту версию, которую скачали.
  ![Путь к Java](/guides/minecraft/java-path.webp)

#### 4.2. 🧠 Выделение памяти (ОЗУ)

- В настройках сборки перейдите в раздел **«Параметры»** → **«Java»**.
- Поставьте галочку **«Память»**.
- В строках **«Минимальный / Максимальный объем памяти»** укажите, сколько оперативной памяти вы хотите выделить (_в мегабайтах_). Рекомендуется ставить одинаковые значения.

> 💡 **Для ванильной сборки рекомендуется от 6000 до 8000 MiB (6–8 ГБ) в зависимости от вашего ПК.**

![Выбор памяти](/guides/minecraft/memory-settings.webp)

#### 4.3. 📝 Аргументы Java

**Аргументы запуска позволяют оптимизировать работу игры и сборщика мусора.**

- В настройках сборки перейдите в раздел **«Параметры»** → **«Java»**.
- Поставьте галочку **«Аргументы Java»**.
- В поле введите следующие аргументы (скопируйте и вставьте):

```text
-XX:+UseG1GC
-XX:+UnlockExperimentalVMOptions
-XX:MaxGCPauseMillis=200
-XX:G1NewSizePercent=20
-XX:G1MaxNewSizePercent=40
-XX:G1HeapRegionSize=16M
-XX:G1ReservePercent=15
-XX:InitiatingHeapOccupancyPercent=20
-XX:+AlwaysPreTouch
```

![Аргументы Java](/guides/minecraft/java-arguments.webp)

- Дополнительные агрументы под ваш ПК можно сгенерировать [**тут**](http://flags.sh)

### 5. 🎛️ Дополнительные настройки

#### 5.1. 🌐 Настройки запуска сборок по умолчанию

- Перейдите в глобальные настройки приложения: на главном экране нажмите иконку **«Параметры»** ⚙️.
- Перейдите во вкладку **Java** и настройте её аналогично пункту **4.1**.
- Теперь все новые сборки будут использовать эту настроенную Java **по умолчанию**.

![Интерфейс Java](/guides/minecraft/java-interface-global.webp)

#### 5.2. 👤 Настройки аккаунта

- Чтобы сменить аккаунт, нажмите на свой никнейм в правом верхнем углу и выберите нужный профиль.
- Чтобы добавить или изменить настройки аккаунта, нажмите **«Управление учетными записями»**.
  - Добавить лицензионный аккаунт: **«Войти через Microsoft»**.
  - Добавить нелицензионный аккаунт: **«Добавить автономную»** учётную запись.

![Управление учётными записями](/guides/minecraft/account-management.webp)

#### 5.3. 👕 Смена скина и плаща

Настроить скины и плащи можно, нажав **«Управление скинами»** в разделе учетных записей.

1.  **Сменить скин:** нажмите на иконку с нужным скином из ранее загруженных.
2.  **Сменить плащ:** выберите нужный из списка доступных вам плащей 🦸.
3.  Кнопка внизу открывает папку со всеми скинами.
4.  **Импорт скина:**
    - 🔗 По прямой ссылке (URL).
    - 👤 По никнейму другого игрока (введите ник и нажмите **«Импортировать игрока»**).
    - 💾 Из файла на вашем компьютере.
5.  Нажмите **«ОК»** и подождите, пока скин загрузится и применится.

![Настройка скина](/guides/minecraft/skin-setup.webp)

#### 5.4. 📂 Группы сборок

Добавить сборку в группу можно, нажав **«Изменить группу»**. Выберите существующую группу или создайте новую, введя её название.

![Имя группы](/guides/minecraft/group-name.webp)

#### 5.5. 📑 Копирование сборки

Любую сборку можно дублировать, перенеся в новую версию те или иные файлы по вашему желанию.

- Нажмите правой кнопкой мыши на сборку и выберите **«Копировать»**.
- В появившемся окне отметьте пункты для переноса в новую копию и нажмите **«ОК»**.

![Копирование сборки](/guides/minecraft/instance-copy.webp)

#### 5.6. 📤 Экспорт сборки

_Freesm поддерживает экспорт вашей сборки со всеми настройками для передачи другим пользователям. При экспорте через **Modrinth** сами моды не включаются в файл (только список ссылок), поэтому итоговый файл весит гораздо меньше._

- Нажмите правой кнопкой мыши по сборке → **Экспортировать** → выберите **Modrinth (.mrpack)**.
- Укажите название и версию сборки.
- Выберите файлы (конфиги, ресурсы), которые хотите включить в пакет.
- Сохраните файл.

![Экспорт сборки](/guides/minecraft/instance-export.webp)

#### 5.7. 🎨 Настройка фона и темы

- Тему приложения можно изменить в меню: **Настройки** → **Launcher** → **Интерфейс**.
  ![Настройка интерфейса](/guides/minecraft/theme-settings.webp)
- Фон можно настроить, следуя [этой инструкции](https://prismlauncher.org/wiki/development/build-instructions/windows) 🖼️.
